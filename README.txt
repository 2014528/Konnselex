Upload both files to your PHP-enabled hosting server in the same folder.

Files:
- index.html: your updated website with the contact form posting to PHP.
- send-mail.php: sends form submissions to contact@konnselex.com.

Important:
- The form will work only on hosting that supports PHP mail().
- If emails do not arrive, check your spam folder or ask your hosting provider to enable mail() / SMTP.
